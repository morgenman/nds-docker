◆ NE_ModelAnimInterpolate()

void NE_ModelAnimInterpolate ( NE_Model *  model,
bool  interpolate 
)

Enables or disables linear interpolation for the drawing of this animated model.

Parameters
modelPointer to the model.
interpolate[true/false] to enable or disable.

If you enable linear interpolation, the animation will be smoother, but it will need more CPU to be drawn. Disable it if your model has a lot of frames and the animation is quite fast. For example, if you change the model frame each vbl this is useless. Default is true.

◆ NE_ModelClone()

void NE_ModelClone ( NE_Model *  dest,
NE_Model *  source 
)

Clone model.

Parameters
destPointer to the destiny model.
sourcePointer to the source model.

NOTE: Be careful with this, if you delete source model and try to draw destiny model game will eventually crash.

You MUST delete destiny model if you delete source model.

The two models MUST BE THE SAME TYPE!!! (Animated or static)

You should use this if you make a tiled floor, for example, or for making lots of enemies.
