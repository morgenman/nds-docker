Texture manipulation functions. A material is just a texture with a color.

◆ NE_TextureSystemReset()

void NE_TextureSystemReset ( int  texture_number,
int  palette_number,
NE_VRAMBankFlags  bank_flags 
)

Resets texture and palette system.

Parameters
texture_numberMax number of textures. If lower than 1, it will create space for NE_DEFAULT_TEXTURES.
palette_numberMax number of palettes. If lower than 1, it will create space for NE_DEFAULT_PALETTES.
bank_flagsVRAM banks where Nitro Engine can allocate textures. In Dual 3D mode, only A and B are available. If you are planning to use a depth BMP, you can only use banks A and B. If you don't tell Nitro Engine to use at least a valid bank, it will use the default ones.
